Pletok
War

Game Design & Developer

2024

Android (Mobile)

Solo game project developed as an undergraduate thesis, focusing on adapting a traditional Indonesian game into a playable digital experience.

The Overview

Pletok War is a top-down action shooter inspired by the traditional Indonesian game Pletokan.
The project explores how a culturally rooted game can be translated into a modern digital format while remaining accessible and fun for contemporary players.
Built using Godot Engine, the game combines simple but engaging mechanics with structured progression, making it suitable for mobile devices and casual play sessions.

Project Showcase — YouTube

The Problem

Traditional games are slowly being forgotten as players shift toward modern digital entertainment. When adapting traditional games into digital form, common challenges include:

  • Translating physical play into engaging digital mechanics
  • Maintaining the identity of the original game
  • Designing gameplay that feels intuitive on mobile platforms

The Solution

I designed Pletok War as a fast-paced, easy-to-learn mobile game while preserving core elements of the original Pletokan concept.

    Key Feature:
  • Top-down shooter gameplay with base-capturing objectives
  • Single-player mode with AI-controlled enemies
  • Health and respawn system to maintain game flow
  • Progressive difficulty across multiple levels
  • Cultural trivia section introducing the background of Pletokan
  • Optimized performance for low to mid-range Android devices

    • The development process followed a structured workflow, allowing rapid iteration and consistent gameplay improvements throughout production.

Tech Stack

Godot Engine GDScript Android

The Outcome

The project resulted in a fully playable Android game developed from initial concept to final build. Through this project, I strengthened my understanding of core game development workflows, including gameplay logic, AI behavior, and mobile performance optimization. User testing showed that the game mechanics were easy to understand and engaging for casual players, validating the design approach and technical implementation. This project also demonstrates my ability to translate a research-based concept into a practical, playable game experience.
