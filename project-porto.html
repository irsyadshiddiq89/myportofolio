A self-initiated project created to represent my skills, design approach, and technical capabilities as a front-end developer.

The Overview

This project is a personal portfolio website built to showcase my work, skills, and design approach as a front-end developer. The goal was to create a clean, modern layout with clear content hierarchy and smooth user experience. The website emphasizes strong typography, balanced spacing, and subtle interactions to guide users through each section naturally. It was developed using semantic HTML, modular CSS, and vanilla JavaScript, with a focus on responsive design and maintainable code structure. This project also serves as a foundation for my future work, allowing me to apply front-end best practices while refining both visual and technical skills.

The Problem

Many personal portfolios focus too much on visual noise or templates, making it difficult to understand the developer’s actual role, thinking process, and technical capability.

The Solution

I designed and developed a personal portfolio website that focuses on clarity, usability, and visual consistency. The solution balances minimalist design with functional interactions, ensuring the content is easy to explore while still feeling polished and professional.

    Key Feature:
  • Clean and responsive layout optimized for desktop and mobile devices
  • Structured section flow (Hero, Work, Contact) for clear storytelling
  • Subtle hover effects and micro-interactions to enhance user experience
  • Modular CSS structure for easier maintenance and scalability
  • Smooth navigation with accessible internal linking

Tech Stack

HTML5 CSS3 (Grid, Flexbox, Custom Properties) Vanilla JavaScript Github Pages

The Outcome

A fully responsive personal portfolio that serves as the foundation for showcasing professional and experimental projects, while reflecting my design approach and front-end development skills.
