Cat Cafe
Website

Front-end Developer

2025

WEB (Desktop/Mobile)

This project was created as a fictional website for a cat café business to demonstrate real-world web design and front-end development skills. The goal was to build a site that could function as a landing page for a café, featuring key pages such as animal profiles, menu, and contact information.

The Overview

The Cat Café Website is a responsive multi-section site designed to showcase a themed café for cat lovers. It introduces visitors to the café’s concept, highlights featured cats, presents the menu, and provides contact details for potential visitors. The layout emphasizes clarity and ease of navigation, with sections that intuitively guide users through the café’s offerings. The project served as an opportunity to apply semantic HTML, flexible CSS layouts, and interactive front-end patterns without relying on large frameworks, ensuring a lightweight and maintainable codebase.

The Problem

The challenge was to create a website that felt like a legitimate business page, despite being a personal project. Instead of a simple static page with filler text, the goal was to simulate realistic content and structure that a café might actually use. This required attention to usability, visual hierarchy, and responsive design so it would work well on both desktop and mobile.

The Solution

To address this, I developed a structured, multi-section website with intuitive navigation and clear content categories. Key features included:

    Key Feature:
  • Responsive layout optimized for desktop and mobile devices
  • Clear navigation with internal links to each major section (Home, Kucing Kami, Menu, Kontak)
  • Sectioned content to introduce the café, animals, menu offerings, and contact information
  • Use of semantic HTML and modern CSS for layout and typography
  • Lightweight JavaScript enhancements for improved interaction

Tech Stack

HTML5 CSS3 JavaScript Github Pages

The Outcome

The Cat Café Website demonstrates the ability to create a professional-feeling business site from scratch, complete with logical section flow, responsive design, and maintainable code. The project showcases practical layout techniques, front-end engineering fundamentals, and a real example of how design and technical execution come together to form a coherent web experience. Visitors can: Easily navigate to learn about the café, View featured cats (simulated “product” content), Browse the menu, Find contact information, As a result, this project strengthens the portfolio by showing capability in building functional, user-focused websites that could be used for real small business presentations.
